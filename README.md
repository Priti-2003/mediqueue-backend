## ✨ Features

- 🔐 **JWT Authentication** — Secure login for Patients, Doctors & Admins
- 🎫 **Smart Token Generation** — Auto-incremented OPD tokens per doctor per day
- 📊 **Live Queue Tracker** — Real-time queue position with 30s auto-refresh
- 📅 **Appointment Booking** — Book appointments with date & symptoms
- 👨‍⚕️ **Doctor Dashboard** — Manage daily patient queue
- 🔔 **Role-Based Access** — Patient / Doctor / Receptionist / Admin
- 🌐 **REST APIs** — Clean API design with proper HTTP methods
